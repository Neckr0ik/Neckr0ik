# Social Engineering Analysis: Psychological Triggers

Analysis of how attackers exploit cognitive biases to bypass technical controls.

## Key Triggers Analyzed

1. **Urgency**: "Your account will be deleted in 2 hours."
2. **Authority**: "Message from the CEO - Urgent Action Required."
3. **Scarcity**: "Only 5 slots left for the mandatory security training."
4. **Social Proof**: "Everyone in HR has already signed the new policy."

## Case Study: Phishing Campaign 2024-12

- **Vector**: Email
- **Lure**: Performance Review Link
- **Trigger**: Fear & Curiosity
- **Success Rate**: 12% Link clicks before training.
